# @jood/v-colrow

테이블 형태의 UI 구현시 열(column)을 기준으로 컴포넌트를 만들기 편하도록 돕기 위한 패키지. \
(= 떨어져 있는 컴포넌트들의 행 높이를 맞춰준다.)
